Audio Steganalysis with CNN

Necessary Package

tensorflow-gpu==1.3, numpy, matplotlib

CNN Architecture

Steganography Algorithm

HCM(Huffman Code Mapping), EECS(Equal Length Entropy Codes Substitution)

Results

Run

Command: python main.py <data_dir> <stego_method> <bitrate> <relative_payload> <model_dir> <log_dir>

data_dir: data dir

mode: train | test

stego_method: EECS | HCM-Gao | HCM-Yan

bitrate: 128 | 192 | 256 | 320

relative_payload:

  • 2 | 3 | 4 | 5 | 6 | 7 | 8 | 9 | 10, if the steganography algorithm is EECS

  • 0.1 | 0.3 | 0.5, otherwise

  • Training

    python main.py -data_dir train EECS 128 2 -model_dir -log_dir

  • Test

    python main.py -data_dir test EECS 128 2 -model_dir
